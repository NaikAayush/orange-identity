# Orange Identity

A digital identity system that makes use of face recognition systems and a permissioned blockchain to streamline the process at airports. Developed as part of [Adani's Jack the Hack](https://www.hackerearth.com/challenges/hackathon/jack-the-hack-hackathon/).

## Code structure

 - [client](./client): Angular based web app.
 - [smart-contract](./smart-contract): Solidity smart contracts, to be deployed on Hyperledger Besu.
 - [api-service](./api-service): Express.js based web API, written in TS. Communicates with database and blockchain.
